# ☕ Sistema de Cafeteria em Java

Este é um projeto simples desenvolvido em Java para simular o atendimento de uma cafeteria. O objetivo principal foi praticar conceitos básicos de lógica de programação e Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades

- Exibição de cardápio dinâmico.
- Seleção de itens pelo usuário via console.
- Acúmulo de itens em um carrinho de compras (Pedido).
- Cálculo automático do valor total da conta.

## 🛠️ Tecnologias Utilizadas

- **Java 25** (ou a versão que você usou)
- **Scanner**: Para leitura de dados no terminal.
- **ArrayList**: Para gerenciamento dinâmico dos itens do pedido.
